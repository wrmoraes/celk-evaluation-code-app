Aconteceu um imprevisto! :-(

Causas comuns para o erro

  1. Você inicou a aplicação pela sua ide e não rodou npm start ou npm run webpack:build.
  2. Aconteceu algum erro na rede enquanto rodava npm install. Se você está por trás de um proxy corporativo, é posível que esse erro tenha sido causado pelo seu proxy. Dê uma olhada nos logs da aplicação, você provavelmente encontrará a causa por lá.
  3. Sua versão instalada do Node.js não funciona bem com o JHipster: favor usar uma versão LTS (Suporte a longo prazo), estas são as únicas suportadas.

Buildando o código do lado do cliente

Se você quer ir rápido, rode ./mvnw para buildar e rodar tudo.

Se você deseja mais controle, com isso debugar o build mais facilmente, você deve seguir os seguintes passos:

  1. Instalar as dependência do npm com o comando npm install
  2. Builda a aplicação cliente com o comando npm run webpack:build ou npm start
  3. Iniciar o servidor com ./mvnw ou usando sua IDE
